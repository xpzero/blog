VSCode中Nodejs使用babel运行ES6模块机制

初始化node​

npm init -y

安装babel模块​

npm i @babel/cli @babel/core @babel/node @babel/preset-env

配置babel文件​

如果目录中没有.babelrc文件，新建一个.babelrc文件。文件内容如下

{
	"presets": [
		"@babel/preset-env"
	]
}

新建测试文件​

创建一个名为test.js的文件。文件内容如下

console.log("babel is ok.");

运行测试文件​

使用Ctrl + `，打开控制台，确保控制台目录在test.js所在目录，运行命令babel-node .\test.js
