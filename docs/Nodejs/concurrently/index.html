concurrently使用记录

可以同时运行多条命令

安装​

npm i concurrently -D
# 或者
npm i concurrently -g

使用​

具体使用参考官方文档

package.json

{
    "scripts": {
        "dev": "concurrently --names \"HTTP,WATCH\" -c \"bgBlue.bold,bgMagenta.bold\"
     \"http-server\" \"npm run watch\"",
        "watch": "..."
    }
}

注意​

concurrently要执行的脚本命令，需全局安装或使用全局包管理工具运行package.json中的脚本命令。

上面代码中的http-server、npm都在nodejs的全局安装路径中

参考

https://www.npmjs.com/package/concurrently
