shell脚本常用指令汇总

遇见一套流程下来好多命令且需要偶尔重复执行的时候，就想写个shell脚本解放双手。这里记录下比较常用的脚本命令

获取执行脚本文件时所在目录​

# dir是变量名，= 前后不能有空格
dir=$(pwd)

# 输入当前文件所在的目录
echo dir

获取执行脚本时其后的参数​

# $0 表示执行当前脚本时的命令
# $1 表示执行脚本时其后的第一个参数
# $2-9 表示执行脚本时其后的第二个到第九个参数

# 执行脚本的命令：./test.sh xxxx

echo "当前执行的命令：$0"
echo "第一个参数：$1"

# 终端换行输出 ./test.sh xxx

获取当前执行脚本命令中脚本的父目录​

# 执行命令：./blog/test.sh

echo "$(dirname $0)"

# 输出：./blog

创建并进入新目录​

# 假设第一个参数为新目录名
dir=$1

cd "$(pwd)/$dir"

# 执行命令时前面要有一个 「.」或者`source` 表示source命令，不让shell创建子shell
# 执行命令： . ./test.sh newdir
